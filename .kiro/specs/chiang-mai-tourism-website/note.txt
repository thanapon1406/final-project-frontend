โปรเจกต์: Redesign เว็บไซต์ห้วยตึงเฒ่า (Huay Tueng Thao Website Redesign)

เว็บไซต์ต้นฉบับ: https://huaytuengthao.com/
- เป็นเว็บไซต์ท่องเที่ยวสำหรับห้วยตึงเฒ่า จังหวัดเชียงใหม่
- ใช้ Bootstrap 5.1.3, Font Awesome 5.10.0, และ Itim font
- มีโครงสร้างที่ดี: navbar, carousel, news marquee, service section, footer
- มีหน้าต่างๆ: หน้าหลัก, ประวัติ, บริการ, ข่าวสาร, เกี่ยวกับเรา, ติดต่อเรา

วัตถุประสงค์โปรเจกต์:
- Redesign เว็บเดิมให้สวยงามและทันสมัยขึ้น
- รักษาเนื้อหาและโครงสร้างเดิมไว้ทั้งหมด
- เอาข้อมูลจากเว็บเดิมมาเก็บใน JSON files
- สร้างระบบ Admin เพื่อแก้ไขข้อมูลใน JSON files

3 จุดที่ควรพัฒนาปรับปรุงในเชิงการออกแบบ:
1. Visual Hierarchy and Content Organization - ปรับปรุงการจัดลำดับความสำคัญของเนื้อหา
2. Color Scheme and Brand Identity - พัฒนาชุดสีที่สะท้อนความงามทางธรรมชาติของห้วยตึงเฒ่า
3. Mobile-First Responsive Design - ออกแบบให้เหมาะกับมือถือและ touch interaction

ข้อกำหนดสำคัญ:
- ใช้ Bootstrap + jQuery เท่านั้น (เหมือนเว็บเดิม)
- เก็บข้อมูลทั้งหมดใน JSON files แยกตามหน้าและหัวข้อ
- มี module API สำหรับการจัดการ JSON files
- มีการใช้ Google Maps แบบ iframe embed (เหมือนเว็บเดิม)
- มี interactive elements ที่ทำให้เว็บน่าสนใจขึ้น

หน้าเว็บที่ต้องมี (เหมือนเว็บเดิมทุกหน้า):
1. หน้าหลัก (index.html) - carousel, services, featured content, news, gallery
2. ประวัติ (page-history.html) - ประวัติความเป็นมาของห้วยตึงเฒ่า
3. บริการ (page-services.html) - กิจกรรมและสิ่งอำนวยความสะดวก
4. ข่าวสาร (page-news.html) - ข่าวประกาศและกิจกรรม
5. เกี่ยวกับเรา (page-about.html) - ข้อมูลองค์กร
6. ติดต่อเรา (page-contact.html) - ที่อยู่, เบอร์โทร, แผนที่
7. หน้า Admin Panel - สำหรับจัดการเนื้อหาทั้งหมด

โครงสร้าง JSON Files:
data/
├── navigation.json (เมนูนำทาง)
├── homepage-carousel.json (carousel 4 slides)
├── homepage-services.json (3-column services)
├── homepage-featured.json (4-column featured content)
├── homepage-news.json (news marquee + updates table)
├── homepage-gallery.json (image gallery)
├── history-content.json (เนื้อหาประวัติ)
├── services-content.json (ข้อมูลบริการ)
├── news-content.json (ข่าวสาร)
├── about-content.json (เกี่ยวกับเรา)
├── contact-content.json (ข้อมูลติดต่อ)
└── footer.json (footer content)

ระบบ Admin ต้องมี:
- ระบบ login แบบง่าย
- แก้ไขเนื้อหาทุกหน้าผ่านฟอร์ม
- อัพโหลดและจัดการรูปภาพ
- แก้ไขไฟล์ JSON โดยตรง
- ระบบ backup และ restore
- เมื่อแก้ไขข้อมูลแล้วหน้าเว็บหลักต้องแสดงข้อมูลใหม่ทันที

เทคโนโลยีที่ใช้ (เหมือนเว็บเดิมทุกอย่าง):
- Bootstrap 5.1.3
- jQuery สำหรับ interactive features
- Font Awesome 5.10.0 สำหรับ icons
- Itim font สำหรับข้อความภาษาไทย
- Google Maps iframe embed
- JSON files สำหรับเก็บข้อมูล

โครงสร้างไฟล์:
- index.html (หน้าหลัก)
- page-history.html (ประวัติ)
- page-services.html (บริการ)
- page-news.html (ข่าวสาร)
- page-about.html (เกี่ยวกับเรา)
- page-contact.html (ติดต่อเรา)
- admin/ (โฟลเดอร์ admin panel)
- data/ (โฟลเดอร์เก็บไฟล์ JSON)
- images/ (โฟลเดอร์รูปภาพ)
- css/ (ไฟล์ CSS ที่ปรับปรุงแล้ว)
- js/ (ไฟล์ JavaScript)

หมายเหตุสำคัญ:
- นี่คือการ REDESIGN เว็บเดิม ไม่ใช่สร้างเว็บใหม่
- ต้องรักษาเนื้อหาและโครงสร้างของเว็บเดิมไว้ทั้งหมด
- เน้นการปรับปรุงด้านการออกแบบให้สวยงามขึ้น
- ทุกข้อมูลต้องมาจากเว็บ huaytuengthao.com

- ทดสอบให้แน่ใจว่าการแก้ไขใน admin จะแสดงผลใน frontend ทันที
